Install with script provided by PiVPN.

Details:

- Wireguard
- Default port 51820
- Custom resolver pointing to homelab DNS.
- Using custom domain

## Add new client

Run `pivpn add` to generate new client config.

Run `pivpn -qr` to regenerate QR code.
